# **웹클라이언트 컴퓨팅 기말 프로젝트 👩🏻‍💻💫**
수업 중에 배웠던 내용을 토대로  ***"HTML, CSS, 자바스크립트"*** 만 사용하여 동적 웹페이지를 만드는 기말 프로젝트입니다.


## **💥 프로젝트 소개**
자율 주제였기 때문에 "자기소개 페이지"를 주제로 하여 나에 대한 웹 사이트를 만드는 것을 목표로 하였습니다. 


## **🌟 개발 기간**

## **⚙️ 개발 환경**
